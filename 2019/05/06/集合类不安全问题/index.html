集合类不安全之List

代码演示，多线程环境List的不安全问题

  • 导致原因
    并发争抢修改导致，参考花名册签名情况。一个同学正在签名，另一个人过来争抢，导致数据不一致异常，并发修改异常。
  • 解决方案
    1. List list = new Vector<>(); 使用Vector，加锁，并发性降低
    2. List list = Collections.synchronizedList(new ArrayList<>()); 使用Collections工具类包装
    3. List list = new CopyOnWriteArrayList<>(); 写时复制
